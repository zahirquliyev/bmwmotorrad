# BMW Motorrad Club Azerbaijan

Official website for BMW Motorrad Club Azerbaijan — a community of passionate BMW motorcycle riders based in Baku, Azerbaijan.

## Features

- Full-screen BMW-style hero slider (3 slides, auto-advance, pause/play, progress bar)
- About the club section
- Upcoming rides & events
- Member profiles
- Photo gallery
- Contact / membership enquiry form

## Tech Stack

- Pure HTML5 / CSS3 / Vanilla JavaScript
- No frameworks or dependencies
- Responsive design
- BMW Motorrad design language inspired UI

## Domain

[bmwmotorrad.az](https://bmwmotorrad.az)

## Deployment

Designed to run inside a Docker container with Nginx + WordPress or as a standalone static page.
